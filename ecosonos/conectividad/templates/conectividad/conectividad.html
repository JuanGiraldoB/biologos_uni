HETEROGENEIDAD ACÚSTICA

Este módulo permite representar sobre un mapa geográfico el valor de una medida (entre 0-1) que se estima para cada punto de muestreo. También realiza la interpolación entre los puntos, de tal manera que se obtiene una capa que representa la medida que se estimó en cada punto y permite comparar la heterogeneidad entre puntos de muestreo.
